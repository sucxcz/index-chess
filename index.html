1. Margin и padding - это свойства в CSS, которые используются для управления пространством вокруг элементов.
Margin определяет внешний отступ элемента от его соседних элементов.
Padding определяет внутренний отступ элемента, который разделяет контент элемента и его границы.
2. Block, inline и inline-block - это значения свойства CSS `display`, которые определяют, как элементы отображаются на веб-странице:
Block элементы занимают всю доступную ширину и начинают новую строку, создавая блочный контейнер.
Inline элементы занимают только столько места, сколько необходимо для их содержимого, и не начинают новую строку.
Inline-block элементы подобны inline, но могут иметь внутренние отступы и располагаться внутри блочных контейнеров.
3. CSS overflow - это свойство, которое определяет, как обрабатывать содержимое элемента, если оно выходит за границы этого элемента. Возможные значения включают `visible` (по умолчанию, содержимое видимо за пределами элемента), `hidden` (скрывает содержимое, выходящее за границы), `scroll` (показывает полосы прокрутки, если содержимое не помещается) и другие.
4. CSS float - это свойство, которое используется для выравнивания элементов внутри контейнера. Элементы, установленные с `float`, будут выравниваться по левому или правому краю контейнера и могут обтекать друг друга.
5. CSS @media rule - это правило в CSS, которое позволяет задавать стили для определенных условий экрана или устройства. Например, вы можете использовать `@media` для применения разных стилей для разных размеров экранов, разрешений или ориентаций устройств.
